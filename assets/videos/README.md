Place your Education tab MP4 files in this folder.

Recommended filenames:
- education-1.mp4
- education-2.mp4
- education-3.mp4
- education-4.mp4

Then update `components/EducationHubScreen.tsx` in the `educationVideos` array.

For hosted files:
- Set each `source` to your direct `.mp4` URL.

For local bundled files:
- Replace each item with `source: require('../assets/videos/education-1.mp4')` style values.
